                                    Super PHP (SP) 
           
Super PHP 框架是一个Super容易上手,Super快速开发, Super专注服务于业务的开发框架。 

#### 1. 总体的架构如下
- 业务层: 专注于业务的操作和开发
- 框架层: 根据现有的开源框架的特点,进行引用。并提供统一的Facade框架服务
- 资源层: 对于不同资源进行封装,包括数据库,缓存, 文件等。
